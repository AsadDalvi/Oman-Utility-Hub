# Oman Smart Utility Hub⚡ ⛽

A responsive dark-mode full-stack bilingual dashboard (English/Arabic) built to calculate monthly rent, utility bills (Nama tariffs), and regional fuel driving costs across the Sultanate of Oman.

## 🌟 Core Features
* **Bilingual Switcher:** Instant click toggle flips layout structure and text readers seamlessly between English (`LTR`) and Arabic (`RTL`).
* **Nama Utility Slabs:** Accurately maps home sizes alongside daily AC run-times to calculate official tiered baisa rates (14 to 32 baisas per unit).
* **Oman Travel Matrix:** Localized kilometer index mapping 12 major hubs (Al Maabilah, Al Khoud, Seeb, Al Khuwair, Ruwi, Barka, Salalah, etc.).
* **Custom Polish Ups:** Bypasses standard month counts to allow up to 44 days for double-shift office commuters, and features a protective alert overlay for `7BHK` inputs.

## 📂 Tech Stack
* **Frontend:** React.js (Vite) + Tailwind CSS
* **Backend:** Node.js + Express.js
