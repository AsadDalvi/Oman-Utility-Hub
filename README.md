# Oman Smart Utility Hub ⚡ ⛽ 💰

A version 2 of updated Oman Smart Utility Hub with responsive full-stack web application which has bilingual dashboard (English/Arabic) built to calculate monthly rent, utility bills (Nama tariffs), regional transit fuel costs, and personal wealth budgets across the Sultanate of Oman.


## 🌟 Core Features
* **Bilingual Switcher:** Instant click toggle flips layout structure and text readers seamlessly between English (`LTR`) and Arabic (`RTL`).
* **Nama Utility Slabs:** Accurately maps home sizes alongside daily AC run-times to calculate official tiered baisa rates (14 to 32 baisas per unit).
* **Oman Travel Matrix:** Localized kilometer index mapping 12 major hubs (Al Maabilah, Al Khoud, Seeb, Al Khuwair, Ruwi, Barka, Salalah, etc.).
* **Salary & Rent Budget Planner:** Computes rent-to-income percentage allocations with a zero-dependency SVG donut chart and high-visibility financial stress alerts.
* **Utility Cost Ledger:** Interactive baseline checklist tracker featuring multi-segment stacked progress bars and dynamic safety ceilings that adapt based on the user's active salary profile.
* **Dual-State Theme Selector:** Integrated global navbar controls to flip the interface canvas instantly between obsidian dark and crisp light themes.
* **Web-Native Mobile Haptics:** Connects browser hardware API channels to emit tactile 15ms click pulses on buttons.


## 📂 Tech Stack
* **Frontend UI:** React.js (Vite) + Tailwind CSS (v4 dark/light layout engine)
* **Backend Brain:** Node.js + Express.js REST API router
* **Deployment Channels:** Client-side files hosted on Vercel and server calculation data processes hosted on Render
